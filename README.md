# Topic Modeling and Sentiment Analysis of Reddit Posts About Depression

## Overview

This project applies Natural Language Processing (NLP) techniques in R to analyze Reddit discussions related to depression.

---

## Objectives

- Clean and preprocess Reddit posts
- Perform tokenization
- Remove stop words
- Analyze word frequencies
- Conduct sentiment analysis
- Perform topic modeling using LDA
- Visualize findings

---

## Dataset

This project uses the **Reddit Depression Dataset** from Hugging Face.

Dataset:
https://huggingface.co/datasets

The dataset is not included in this repository because of its size. All analyses can be reproduced by downloading the dataset from Hugging Face.

## Tools

- R
- tidyverse
- tidytext
- topicmodels
- ggplot2
- wordcloud

## Key Findings

The analysis suggests that Reddit discussions about depression frequently revolve around emotional distress, loneliness, relationships, and family. Topic modeling identified multiple discussion themes, while sentiment analysis demonstrated the predominance of negative emotions across posts.
